### 트리란?

#### 트리의 순회(Tree Traverse) 과정
- 전위 순회(preorder): 뿌리(root), 혹은 방문 정점 노드 우선 방문  
- 중위 순회(inorder): 왼쪽 하위 트리, 혹은 자식 노드 방문 후 뿌리(root)를 방문  
- 후위 순회(postorder): 하위 트리 모두 방문 후 뿌리(root)를 방문
